
## Run Locally

Clone the project

```bash
  git clone https://github.com/zagdiablo/bm-grade-answersheet.git
```

Go to the project directory

```bash
  cd /into/your/directory/bm-grade-answersheet
```

Install dependencies

```bash
  pip install -r requirements.txt
```

Start the server

```bash
  python app.py
```
